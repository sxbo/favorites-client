"#favorites-client" 

# 基于webpack打包的favorites项目前端代码
# 技术栈：react+react-router+react-redux+antd+axios
 
### usage：

1.	`git clone https://github.com/sxbo/favorites-client.git <dir-name>`
2.	`cd <dir-name>`
3.	`cnpm i` or `npm i`

### run in dev
  `npm run dev`
### build
  `npm run build`
